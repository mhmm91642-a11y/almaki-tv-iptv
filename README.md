# ALMAKI TV | اشتراكات IPTV الفاخرة

صفحة هبوط احترافية لعرض باقات اشتراك IPTV لموقع ALMAKI TV.

## المميزات
- تصميم عصري وفاخر باللونين الأسود والذهبي.
- متوافق مع جميع الشاشات (Responsive Design).
- عرض منظم لمختلف السيرفرات (Everest, Hulk, Strong, Falcon, Spider).
- روابط مباشرة للطلب عبر WhatsApp.
- أقسام توضيحية لخطوات التفعيل ومميزات الخدمة.

## التقنيات المستخدمة
- HTML5
- CSS3 (Custom Styles)
- Google Fonts (Cairo)

## كيفية الاستخدام
يمكنك فتح ملف `index.html` في أي متصفح ويب لعرض الصفحة.
